# 29단계 카테고리 수동 추가 전용화

## 변경 배경
- 이미지 업로드/AI 분석 이후 감지된 카테고리가 `카테고리 항목` 목록에 자동으로 나타나는 동작이 있었다.
- 요구사항은 `카테고리 항목`이 사용자 직접 추가 시에만 반영되는 것이다.

## 변경 파일
- `src/pages/Editor.tsx`
- `src/components/editor/EditorRightPanel.tsx`

## 변경 내용
1. 카테고리 드롭다운 옵션과 카테고리 항목 목록을 분리했다.
- 드롭다운(`categoryOptions`)은 현재 선택된 카테고리를 계속 표시한다.
- 카테고리 항목 목록(`categoryItems`)은 저장된 수동 항목만 표시한다.

2. 중복 체크 기준을 수동 항목 목록으로 제한했다.
- `addCustomCategory`에서 중복 판정 대상을 드롭다운 전체가 아니라 수동 항목 목록으로 변경했다.
- 따라서 업로드/분석으로 일시 반영된 값은 자동 등록으로 간주되지 않는다.

3. 삭제 로직 대상을 수동 항목 목록으로 제한했다.
- `removeCategory`가 수동 항목만 삭제하도록 변경했다.

## 기대 동작
- 이미지 업로드 후 AI가 카테고리를 바꿔도 `카테고리 항목` 목록에는 자동 추가되지 않는다.
- 사용자가 입력창 + `추가` 버튼으로 등록한 항목만 목록에 남는다.
